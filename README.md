## js関数（v3）

```js
toast(message)
// トーストを表示します

showMenu(n)
// メニューn番目を表示します (例: showMenu(1) → 1番目)

isValidNumber()
// 入力欄の検証（デフォルトで値のみ）

getScriptName(scriptNumber)
// スクリプトの名前を取得します

main(scriptNumber)
// runを呼び出します
```
